# Doomsday
Interactive, browser-only app for learning and using John Conway's Doomsday algorithm.

## Quick start
1. Open `src/index.html` in a modern browser.
2. Enter a date between `1700-01-01` and `2100-12-31`.
3. Switch between `Beginner` and `Expert` modes for different explanation depth.
4. Use the reference panel and month view to see monthly doomsday anchors and target-day offset context.

## Run tests
Requires Node.js 18+.

```bash
node tests/doomsday.test.mjs
```
